Data Customizations




Lifestyle and Home Corp is striving to provide their customers a more personalized experience allowing them to add special delivery instructions when placing orders.

There must also be an option to save these delivery instructions in the customer’s profile and use them as default for all future orders.

Customers who opt in to receive promotional newsletter must be added to a list maintained in a Custom Object.

Steps



Perform the following programmatically, using the SDK:
  1. Create a Custom Type to update delivery instructions in Orders with an option to save them in Customer’s profile
  2. Create a Custom Object to store users emails when they subscribe to promotional newsletter.


Perform the following steps in the Merchant Center:
  1. Inspect the order for custom delivery instructions.
