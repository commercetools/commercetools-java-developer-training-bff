Our Use Case: Lifestyle and Home (L&H) Corp


You are working for Lifestyle and Home Corp. The user wants to search for products, and filter them by attributes like color, finish as well as price range.

  1. The user can search through products by keyword.
  2. The system retrieves the relevant products along with filter choices.
  3. The system displays the filtered search results based on user selections.
  4. The user can view the product details with an option to add the products to the cart.
